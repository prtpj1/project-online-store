## Project Description
In this project, we created a simplified version of an online store, without database persistence.  

This was my first group project that I helped develop while studying at Trybe. A challenge spiced with anxiety.  

Would it work out?  
Would everyone collaborate?  
Would we be able to handle any issues that might arise?  

I confess I was quite nervous, but everything went smoothly. It was during this project that I saw different approaches to problems with different solutions, where I learned a lot and was also able to share what I knew _(I learned more than I taught ü§ì)_.  

We developed the features together according to tasks defined on a Kanban board.  
The use of Slack and Zoom was also crucial for the team's communication.  

Much of the development was done in a style similar to pair programming, but with all of us helping each other.  

At other times, each member wrote some parts of the code independently, along with corrections and small modifications.  

The group had colleagues of varying levels and knowledge, and we developed by contributing in the best way each of us could.
